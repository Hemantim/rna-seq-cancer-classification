# RNA-Seq Cancer Classification
Machine learning pipeline for multi-class cancer classification using RNA-Seq gene expression data. 

## Project Overview
This project implements a complete machine learning pipeline for multi-class cancer classification using RNA-Seq gene expression data. The objective was to develop a structured and reproducible workflow for preprocessing high-dimensional biological data and evaluating multiple classification models.

## Problem Statement
Gene expression datasets are high-dimensional and require careful preprocessing, feature selection, and model evaluation. The goal was to classify cancer types based on RNA-Seq expression profiles using supervised learning techniques.
